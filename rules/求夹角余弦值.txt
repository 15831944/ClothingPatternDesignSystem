规则 求夹角余弦值(直线1上端点，直线1上另一端点，直线2上端点，直线2上另一端点)
规则 第1个点与第3个点重合，第2个点与第4个点重合。
规则 直线1与直线2的长度相等。
规则 输出两直线之间的夹角的余弦值。

输入 点 a1
输入 点 a2
输入 点 b1
输入 点 b2

点 v = 方向向量(b1, a1)
参数 dx = v.横坐标
参数 dy = v.纵坐标

参数 x1 = a2.横坐标
参数 y1 = a2.纵坐标
参数 x2 = b2.横坐标
参数 y2 = b2.纵坐标
参数 x3 = x2 + dx
参数 y3 = y2 + dy

参数 cos = (x1 * x3 + y1 * y3) / (x1 * x1 + y1 * y1)

输出 cos
