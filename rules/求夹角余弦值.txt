规则 求夹角余弦值(直线1上端点，直线1上另一端点，直线2上端点，直线2上另一端点)
规则 将参数中的第1个点与第3个点重合，作为角的顶点。
规则 直线1与直线2的长度相等。
规则 输出两直线之间的夹角的余弦值。

输入 点 a1
输入 点 a2
输入 点 b1
输入 点 b2

参数 a1x = a1.横坐标
参数 a1y = a1.纵坐标
参数 b1x = b1.横坐标
参数 b1y = b1.纵坐标
参数 a2x = a2.横坐标
参数 a2y = a2.纵坐标
参数 b2x = b2.横坐标
参数 b2y = b2.纵坐标

参数 xa = a1x - a2x
参数 ya = a1y - a2y
参数 xb = b1x - b2x
参数 yb = b1y - b2y

参数 la = xa * xa + ya * ya
la = sqrt(la)
参数 lb = xb * xb + yb * yb
lb = sqrt(lb)

参数 cos = (xa * xb + ya * yb) / (la * lb)

输出 cos
